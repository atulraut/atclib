Assignment Topic: Arrays 1-D

1)	Accept marks of 6 subjects & find the average. Print the marks as well as average.

2)	Find maximum and minimum of 10 integers. Print the result along with the numbers entered.

3)	Search a given number within an array using Sequential Search.

4)	Search a given number within an array using Binary Search.

5)	Write a program to reverse array. User will provide element count.

6)	Accept 10 elements from user. Then delete all duplicated elements & print them.

7)	Write a program to calculate mode of the given set of elements.
Mode of a set of elements is defined as the data element that is occurred maximum times in the set.
